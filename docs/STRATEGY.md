# qkquant 策略文档

## 策略概览

| 策略 | 类型 | 持股数 | 持仓周期 | 适合市场 |
|------|------|--------|----------|----------|
| **momentum** ⭐ | 绝对动量趋势跟踪 | ≤5 | 数周~数月 | 趋势市 |
| **momentum_breakout** | 动量突破（强化追高） | ≤8 | 数周~数月 | 强趋势市 |
| **ma_boll** | 双均线 + 布林带 | ≤5 | 1~4 周 | 震荡偏多 |
| **ma_breakout** | 双均线金叉 | ≤10 | 1~4 周 | 趋势启动期 |
| **relative_strength** | 横截面相对强度 | 10 | 20 日调仓 | 分化市 |

> ⭐ momentum 是主力策略，回测充分。

---

## 一、momentum（绝对动量趋势跟踪）

### 逻辑

```
买入：过去 20 日涨幅 > +3%，且当前价距 20 日高点 < 10%（紧贴趋势）
卖出：过去 20 日涨幅 < -3%（动量反转）
```

不追高、不抄底，只在趋势已经形成且尚未衰竭时入场。

### 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `mom_window` | 20 | 动量计算窗口（日） |
| `entry_threshold` | 0.03 | 入场门槛：20 日涨幅 > +3% |
| `exit_threshold` | -0.03 | 出场信号：20 日涨幅 < -3% |
| `drawdown_from_peak` | 0.10 | 距 20 日高点不能超过 10% |
| `max_positions` | 5 | 同时最大持仓数 |
| `min_price` | 1.0 | 最低股价过滤 |
| `min_float_cap` | 20 亿 | 最小流通市值（过滤庄股） |
| `min_amount` | 500 万 | 最小日成交额（过滤无流动性） |

### 回测结果

```
周期:   2023-01-01 ~ 2026-06-13
股票池: HS300（剔除 ST）
本金:   30,000
复权:   前复权 (qfq)
----------------------------------
最终值:   36,704
收益率:   +22.35%
最大回撤: 16.2%
年化:     ~5.9%
```

## 二、momentum_breakout（动量突破）

### 与 momentum 的区别

| | momentum | momentum_breakout |
|---|---|---|
| 入场门槛 | +3% | **+5%**（更强动量） |
| 距高点限制 | 10% | **3%**（必须紧贴峰值） |
| 创新高要求 | 无 | **必须创 10 日收盘新高** |
| 最大持仓 | 5 | 8 |

设计意图：在 A 股追真正的突破——"等回调"往往等来的是转势，不如确认突破时追入。

---

## 三、ma_boll（双均线 + 布林带）

### 逻辑

```
买入：5 日线金叉 20 日线 + 收盘价在布林中轨上方 + 不靠近上轨（< 上轨×0.97）
卖出：5 日线死叉 20 日线 或 收盘跌破布林中轨×0.95
```

金叉确认短期趋势向上，布林带限制追高（不在上轨附近买）和止损（跌破中轨出）。

### 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `fast` / `slow` | 5 / 20 | 快慢均线周期 |
| `boll_period` | 20 | 布林带周期 |
| `boll_dev` | 2.0 | 标准差倍数 |
| `upper_buffer` | 0.03 | 距上轨 3% 缓冲（不追高） |
| `mid_break_pct` | 0.05 | 跌破中轨 5% 止损 |
| `cross_lookback` | 2 | 金叉后 2 天内仍可入场 |

---

## 四、ma_breakout（双均线突破）

### 逻辑

```
买入：5 日线上穿 20 日线 + 价格站上 MA60 + 当日放量（> 5日均量）
卖出：5 日线下穿 20 日线 或 跌破 MA20×(1-5%)
```

比 ma_boll 更纯粹——不依赖布林带，但加了 MA60 趋势过滤和放量确认，适合趋势启动初期。

### 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `fast` / `slow` | 5 / 20 | 快慢均线周期 |
| `stop_loss_pct` | 0.05 | 跌破慢线 5% 止损 |
| `trend_period` | 60 | 趋势过滤 MA 周期 |
| `trend_filter` | true | 价格必须站上 MA60 |
| `volume_filter` | true | 当日量必须 > 5日均量 |

## 五、relative_strength（相对强度）

### 逻辑

横截面策略：每 20 个交易日对所有 HS300 成分股按过去 60 日涨幅排序，持有 top 10。跌破前 10 则替换。

与前面四个时序策略不同——它不关心绝对涨跌，只做"强者恒强"的横截面轮动。

## 六、风控体系

所有策略共用同一套风控规则，在策略 yaml 的 `risk:` 段配置。

### 规则一览

| 规则 | 作用 | momentum 默认 | 说明 |
|------|------|:---:|------|
| `position_stop` | 单票硬止损 | **8%** | 从入场价跌超阈值即强平 |
| `trailing_stop` | 单票移动止盈 | **10%** | 从持仓最高价回撤超阈值强平 |
| `concentration` | 集中度上限 | **22%** | 单票市值占比上限 |
| `portfolio_drawdown` | 组合熔断 | **15%** | 组合从峰值回撤超阈值全平 + 20 天冷却 |
| `blacklist` | 黑名单过滤 | 关闭 | 手动排除特定代码 |

### 风控收紧前后对比（momentum）

| | 旧风控 | 新风控 | 变化 |
|---|---|---|---|
| 硬止损 | 关闭 | **8%** | 新增 |
| 移动止盈 | 12% | **10%** | 收紧 |
| 组合熔断 | 20% | **15%** | 收紧 |
| **最大回撤** | 20.8% | **16.2%** | -4.6pp |
| **收益率** | +16.19% | **+22.35%** | +6.2pp |

收紧风控同时提升了收益和降低了回撤——烂票被及时砍掉，保住了本金参与下一轮机会。

## 七、趋势过滤体系

### 7.1 ADX 趋势强度过滤

```
ADX < 20 → 震荡市 → 禁止买入
ADX ≥ 20 → 趋势市 → 允许买入
```

ADX（Average Directional Index）衡量趋势强度，不判断方向。低于 20 表示市场没有方向（震荡/盘整），是趋势跟踪策略的"绞肉机"。

### 7.2 市场自适应

```
HS300 收盘 > MA60 → 多头排列 → 关闭 ADX 过滤（充分参与趋势）
HS300 收盘 ≤ MA60 → 空头排列 → 开启 ADX 过滤（震荡/下跌市谨慎入场）
```

参数：`adx_market_filter: true` 启用此行为。

### 7.3 重新入场冷却期

```
卖出某只票 → 20 天内不重新买入同一只票
```

防止震荡市中反复进出同一只票（whipsaw）。参数：`cooldown_days: 20`，设为 0 关闭。

### 三层过滤效果（信号绩效 2026-05-07 ~ 06-12，震荡市）

| 过滤层 | 交易数 | 澜起科技进出 | 动量胜率 |
|--------|:-----:|:----------:|:-------:|
| 无过滤 | 96 | 5 次（4 亏 1 盈） | 18%（3/17） |
| +ADX | 67 | 5 次 | 18%（3/17） |
| +ADX + 冷却期 | **48** | **1 次（+6.7%）** | **50%（3/6）** |

## 八、日常使用

### 数据更新
```bash
qkquant update-data
```

### 信号扫描
```bash
# 裸信号模式（推荐，不受模拟仓影响）
qkquant scan --raw

# 推送到微信/企微/钉钉
qkquant scan --raw --push
```

### 持仓跟踪
```bash
qkquant track --push
```

### 回测
```bash
# 单策略回测
qkquant backtest momentum --start 2023-01-01

# 多策略对比
qkquant compare momentum ma_boll --start 2023-01-01
```

### 信号绩效复盘
```bash
python scripts/signal_performance.py
```
输出 `reports/signal_performance_YYYY-MM-DD.md`，模拟"完全按推送操作"的每笔买卖收益。

## 十、策略参数速查

### 通用过滤参数（所有时序策略）

| 参数 | 默认值 | 说明 |
|------|:-----:|------|
| `adx_threshold` | 20 | ADX 趋势过滤阈值，0=关闭 |
| `cooldown_days` | 20 | 卖出后冷却天数，0=关闭 |
| `adx_market_filter` | true | 多头市自动关闭 ADX |

### 风控参数建议

| 市场环境 | position_stop | trailing_stop | portfolio_drawdown |
|----------|:-----------:|:------------:|:------------------:|
| 当前（稳健） | 8% | 10% | 15% |
| 激进（牛市） | 关闭 | 15% | 25% |
| 保守（熊市） | 5% | 8% | 10% |
